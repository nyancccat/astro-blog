## 字体显示测试 Font Display Test

### 中文段落测试

这是一段纯中文的测试内容。我们使用 hyksk 字体来显示中文字符，看看在不同字重下的显示效果如何。中文字体的粗细、间距和整体视觉效果都很重要。

在导航栏中，中文字体的大小和英文字体应该保持视觉上的平衡。如果中文字体显得太小，可能需要调整字体权重或者字号。

### English Paragraph Test

This is a pure English paragraph for testing. We use ZenMaruGothic-Medium font for English characters. The font weight, spacing, and overall visual effect are important for readability.

In the navigation bar, the size of Chinese and English fonts should maintain visual balance. If the Chinese font appears too small, we may need to adjust the font weight or font size.

这是一个中英文混合的段落。When we mix Chinese and English in the same paragraph, 字体的切换应该是平滑的。The font switching should be smooth. 我们需要确保 both languages are displayed properly 在同一行中。
