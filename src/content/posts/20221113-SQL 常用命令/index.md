## DDL 数据定义

### 一、库管理

#### 1.查看数据库

```sql
show databases
```

#### 2.使用数据库

```sql
use 数据库名称
```

#### 3.创建数据库

```sql
create database 数据库名称 [charset utf8]
```

`charset utf8 为可选`。

#### 4.删除数据库

```sql
drop database 数据库名称
```
